## הוראות כלליות (Generic Instructions)

1.  **התחבר לחשבון שלך** באתר שבו רכשת את הדומיין `ai-experts.co.il`.
2.  מצא את אזור ניהול הדומיין (Domain Management / DNS Settings).
3.  חפש אפשרות בשם **Email Forwarding** או **Email Aliases**.
4.  צור הפניה חדשה (Add Forwarder):
    *   **Forward from (מאת):** `info` (או הכתובת המלאה `info@ai-experts.co.il`)

5.  שמור את השינויים.
